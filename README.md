# Hotel Management System
**Developer:** Maira, Hiba Malick, Zoya Siddiqui, Waniya Azhar  

## Project Overview
This is a Python-based desktop application designed to manage hotel operations, including guest registration, room inventory, and automated billing. It uses a Tkinter GUI and a MySQL database backend.

## Dependencies
Before running the application, ensure you have the following external libraries installed:
* **Pillow**: For image processing.
* **mysql-connector-python**: For database connectivity.

### Installation Command
```bash
pip install Pillow mysql-connector-python

How to Run the Application
1. Clone this repository to your local machine.
2. Ensure you have a MySQL server running.
3. Update the config.py file with your local MySQL credentials.
4. Run the main application file:

python hotel.py
