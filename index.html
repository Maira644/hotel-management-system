Hotel Management System

University Project | Version Control & Deployment

Project Overview

This application is a comprehensive Hotel Management System developed using Python. It utilizes the Tkinter library for a user-friendly Graphical User Interface and MySQL for robust database management. The system is designed to streamline administrative tasks in a hospitality environment.

Core Functionalities

Admin Authentication:
Secure login system to restrict access to authorized personnel only.
Customer Management:
Handles guest registration and profile persistence in the database.
Room Inventory:
Allows administrators to add, update, and manage room types and availability.
Booking & Billing:
Calculates stay costs automatically and manages check-in records.

Technical Stack

  • Frontend: Python Tkinter
  • Backend: Python 3.x
  • Database: MySQL
  • Version Control: GitHub
